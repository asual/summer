This application demonstrates the capabilities of the Summer presentation library by allowing you to manage a list of technologies and their properties. It utilizes most of the key features provided by the library such as:
  • HTML5 markup and templating
  • Declarative AJAX regions
  • Data binding and error reporting
  • REST and content negotiation support
  • Bean validation and type conversion
  • Servlet 2.5 and Google AppEngine support
  • Packaging and minification of CSS and JavaScript files
  • Form components with uniform custom look & feel
  • Leaner CSS support through less.js
  • Beautiful markup output
  • Support for web fonts
